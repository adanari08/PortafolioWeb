# Descripción del proyecto

Este proyecto consiste en el desarrollo de un portafolio web personal como parte de la materia de Programación Web.

Para su elaboración se utilizó una plantilla existente basada en **Bootstrap**, la cual fue personalizada con mi información personal, habilidades, proyectos académicos y datos de contacto.

El objetivo fue adaptar una plantilla profesional para presentar de manera organizada mi perfil como estudiante de Ingeniería en Sistemas Computacionales.

# Proceso de creación

Para desarrollar este portafolio se siguieron los siguientes pasos:

### 1. Selección de la plantilla

Se eligió la plantilla **iPortfolio**, desarrollada con Bootstrap, debido a su diseño moderno y adaptable.

### 2. Descarga de la plantilla

Se descargó la plantilla desde su sitio oficial.

### 3. Personalización

Se reemplazó toda la información de ejemplo por información personal.

Se modificaron:

- Nombre
- Fotografía
- Descripción personal
- Información de contacto
- Habilidades
- Educación
- Idiomas
- Proyectos

### 4. Modificación del diseño

Se realizaron cambios en los colores de la plantilla para darle una apariencia más sobria utilizando tonos café y beige.

También se tradujeron las secciones al español.

### 5. Adaptación del contenido

Se agregaron proyectos académicos reales desarrollados durante el semestre.

Se eliminaron secciones que no eran necesarias, como testimonios y estadísticas.

### 6. Pruebas

Finalmente se verificó el funcionamiento del menú, las imágenes, el formulario y el diseño responsivo en el navegador.
